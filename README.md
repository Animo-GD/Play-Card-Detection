# How To Install
1. Create a virtual environment
   ```bash
   python -m venv play-card
   ```
2. Activate the virtual environment
   ```bash
   ./play-card/Scripts/activate
   ```
3. Install requirements
   ```bash
   pip install -r requirement.txt
   ```
4. Open `main.py` and run it.
